## 🌬️ ReneWind Failure Prediction (ML Project)

### 📌 Project Overview

This project was completed as part of the **Advanced Machine Learning program (PGP-AIML)** at Great Learning.

The goal is to predict wind turbine generator failures using sensor data, enabling proactive maintenance and reducing operational costs.

---

### 🎯 Objectives

* Analyze sensor data from wind turbines
* Build classification models to predict failures
* Minimize false negatives (critical failures missed)
* Improve maintenance planning and cost efficiency

---

### 🛠️ Tools & Technologies

* Python
* NumPy
* Pandas
* Matplotlib & Seaborn
* Scikit-learn
* Neural Networks

---

### 📊 Project Workflow

#### 🔹 Exploratory Data Analysis (EDA)

* Sensor data distribution
* Correlation analysis
* Outlier detection

#### 🔹 Data Preprocessing

* Handling missing values
* Feature scaling
* Data balancing

#### 🔹 Model Building

* Logistic Regression
* Decision Tree
* Random Forest
* Neural Networks

#### 🔹 Model Evaluation

* Accuracy
* Precision & Recall
* F1 Score
* Focus on reducing false negatives

---

### 📈 Key Insights

* Certain sensor patterns strongly indicate potential failures
* Neural networks improve prediction accuracy
* Reducing false negatives is critical for avoiding breakdowns
* Early detection significantly reduces maintenance costs

---

### 💡 Business Recommendations

* Implement predictive maintenance using ML models
* Monitor high-risk turbines continuously
* Schedule maintenance before failure occurs
* Use real-time sensor data for better decision-making
